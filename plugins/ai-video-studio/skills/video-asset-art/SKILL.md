# 参考资产美术指导

## 目标

剧本完成后、写视频提示词之前：先锁定**全片视觉一致性**资产，并给出可交给图像模型的提示词。默认中文。

详细清单与优先级见 `references/image-asset-planning.md`。

## 硬规则

1. 有剧本时，**禁止**跳过资产直接写长视频提示词。  
2. 默认图像模型：`GPT Image 2`；复杂多图/强文字可建议 `Nano Banana Pro`。  
3. 资产优先级：定妆照 → 三视图 → 道具/产品 → 场景 → 风格帧 → 首尾帧。  

## 工作步骤

1. 从剧本提炼：人物、服装、道具、场景、视觉符号、风格。  
2. 列关键资产表（用途、是否必须图生、是否需要多角度）。  
3. 输出定妆照 / 三视图 / 道具 / 场景 / 风格帧提示词（可直接给图像模型）。  
4. 标注哪些序列应图生视频、首帧、首尾帧或多参考图，并说明每张参考图用途。  
5. 给出一致性约束：禁止变化项（脸、服装、比例、标志色等）。  

## 默认输出

```text
## 资产总表
（名称 / 类型 / 用途 / 建议生成方式 / 优先级）

## 定妆与角色
（提示词 + 生成参数建议）

## 道具与产品

## 场景与风格帧

## 首尾帧规划
（对应序列、用途）

## 图生视频参考绑定建议
（序列号 → 使用哪些资产图）
```

## 与其它 Skill 的衔接

- 无剧本 → `video-script-writer`  
- 资产齐后 → `video-sequence-planner` / `video-prompt-engineer`  
- 真要出图 → 可建议 ImageGenerator 等执行类插件  

## 边界

- 不直接调用图像 API，除非用户提供可用工具。  
- 不生成侵权肖像、规避审核或违规内容方案。
