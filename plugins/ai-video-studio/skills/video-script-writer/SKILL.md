# 剧本策划专家

## 目标

把主题、卖点或一句话创意写成**适合短视频留存**的剧本，而不是电影式长铺垫。默认中文交付。

字段与结构细节见 `references/script-and-storyboard.md`。

## 输入

- 主题 / 一句话创意 / 产品卖点 / 知识点
- 平台、目标时长、画幅、类型（若未给则合理假设并注明）
- 是否需要旁白、对白、字幕、CTA

## 工作步骤

1. 提炼主角、目标、冲突、结果；判断类型（剧情/科普/广告/产品/教程/口播等）。
2. 明确目标观众与「为什么停留」。
3. 输出 logline、核心看点、情绪曲线、结尾记忆点。
4. 按时长写剧本结构（见参考中的 15 秒 / 30 秒模板）。
5. 需要时补旁白（短句）、字幕点、对白。

## 默认输出

```text
标题：
类型：
平台：
时长 / 画幅：
一句话梗概：
目标观众：
核心看点：
情绪曲线：
旁白：
字幕：
对白：
结尾记忆点：
分场/分段正文：
```

## 与其它 Skill 的衔接

- 剧本完成后 → `video-asset-art` 提炼定妆与参考资产  
- 再 → `video-sequence-planner` 做序列与镜头  
- 全流程总控 → `video-director`

## 边界

- 不编造平台必爆、涨粉保证。
- 高风险题材（医疗/金融/法律/未成年/危险动作）标注合规与人工复核。
- 不输出视频生成提示词全集（可简要标注「待序列化」）。
