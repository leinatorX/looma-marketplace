# 序列分镜导演

## 目标

把已有剧本（或简要剧情）拆成 AI 可生成的**序列**与成片可剪的**镜头**。默认中文。

必读参考：

- `references/sequence-planning.md` — 层级与时长合法化  
- `references/shot-language.md` — 景别、构图、运镜表达  

## 核心规则

```text
集 / 整条视频
  -> 序列（AI 视频生成单位）
    -> 镜头（成片剪辑单位）
```

- 视频生成提示词默认按**序列**准备，不给每个 1–3 秒镜头单独写生成提示词。
- 生成时长必须满足模型限制；短镜头从合法序列中**裁切**。
- 默认视频模型：`Grok Imagine Video`（用户指定优先）。

## 工作步骤

1. 确认平台、成片总时长、画幅、目标视频模型。  
2. 按场景/动作链/参考资产组切序列。  
3. 为每序列标注：生成方式（文生/图生/首帧/首尾帧/多参考图）、推荐模型、合法生成时长、参考资产、首尾帧需求。  
4. 在序列内规划镜头：成片时长、使用区间、景别、构图、运镜、声音点、字幕点。  
5. 输出序列规划表 + 镜头剪辑表。  

## 序列表字段

```text
序列号 / 剧情功能 / 覆盖镜头 / 场景 / 角色 / 关键道具 / 动作链 / 情绪目标 /
推荐生成方式 / 推荐模型 / 合法生成时长 / 参考资产 / 首帧 / 尾帧 / 可剪辑区间 / 风险
```

## 镜头表字段

```text
镜头号 / 成片时长 / 所属序列 / 使用区间 / 画面功能 / 画面内容 /
景别 / 构图 / 运镜 / 声音点 / 字幕点 / 剪辑备注
```

## 与其它 Skill 的衔接

- 缺剧本 → 先 `video-script-writer`  
- 缺定妆/参考图规划 → `video-asset-art`  
- 需要可投模型的提示词正文 → `video-prompt-engineer`  

## 边界

- 模型时长限制变化快；用户要「最新」时提示联网复核官方文档。  
- 不虚构未确认的模型参数。
