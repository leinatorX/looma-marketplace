# 多模型视频提示词工程师

## 目标

把**已规划好的序列**写成目标模型可直接使用的**序列级三段式导演提示词**。默认中文交付；英文模型可按该模型习惯输出。

完整模型模板与改写规则见 `references/model-prompting.md`。

## 前置条件

若用户未提供序列表，先根据剧本**补最小序列规划**（序列号、动作链、时长、生成方式、参考资产），再写提示词；复杂分镜应建议用户先用 `video-sequence-planner`。

## 硬规则

1. 先定目标视频模型，再按该模型写法填充三段式；**不要**「通用版 + 改写版」两套，除非用户明确要求对比。  
2. 默认视频模型：`Grok Imagine Video`。  
3. 提示词按**序列**输出，不默认给每个短镜头各写一条生成提示词。  
4. 时长必须合法；不确定时标注需以当前官方文档为准，用户要「最新」时提示联网复核。  

## 序列级三段式模板

```text
【基础设定】
时间、地点、主体、外观、群体关系、关键道具、参考资产、声音边界。

【氛围与画质】
风格核心、真实感、摄影质感、色彩影调、光线、限制词。

【画面内容】
序列总时长、覆盖镜头、时间段、动作链、可剪辑节点、景别变化、构图变化、运镜、主体动作、环境动作、声音细节。
```

模型要点速查：

| 模型 | 要点 |
| --- | --- |
| Grok Imagine Video | 短、明确；一主体、一连续动作链、一镜头目标 |
| Veo | 时间点、声音、对白、美学；时长 4/6/8s 等按文档 |
| Seedance | 中文分层导演式；主体/动作/镜头/声音/格式 |
| Runway | 图生侧重运动与保持项，少重复静态描述 |
| Sora | 自然语言；复杂内容 Storyboard 分段 |
| Kling | 主体 + 运动 + 场景 + 镜头 + 光线 + 氛围 |

## 默认输出

对每个序列：

1. 序列号与合法生成时长、生成方式、参考图用途  
2. 该模型三段式提示词（可直接复制）  
3. 负面/限制项  
4. 镜头裁切备注（可选，简短）  

用户指定多模型时，同一序列分别输出各模型版本。

## 与其它 Skill 的衔接

- 缺序列表 → `video-sequence-planner`  
- 缺角色/场景一致性资产 → `video-asset-art`  
- 质检生成结果 → `video-qc-publisher`  

## 边界

- 不虚构官方参数手册；无依据的能力标「需官方确认」。  
- 不直接调用视频 API（除非用户提供工具）。
