# AI 视频生成质检

## 质检目标

判断生成视频是否可进入剪辑，还是需要重生成、局部修改或调整分镜。

## 快速结论等级

```text
可用：主体、动作、镜头、风格基本符合，可直接剪辑。
可修：主体或镜头大体可用，但有小穿帮、节奏或细节问题。
重生成：核心主体、动作、镜头或场景失败。
弃用：存在严重变形、违规风险、不可剪辑问题。
```

## 检查维度

### 参考资产一致性

- 是否先完成了主要人物定妆照、道具图、场景图或风格帧。
- 需要图生视频的序列是否实际使用了参考图。
- 首帧和尾帧是否能自然衔接，角色、服装、场景和镜头方向是否一致。
- 多参考图是否明确区分人物、道具、场景和风格用途。

### 序列与时长合法性

- 是否按 `集 -> 序列 -> 镜头` 输出。
- 视频生成提示词是否按序列输出，而不是给每个短镜头各写一条。
- 每个序列的生成时长是否满足目标模型限制。
- 镜头是否标注所属序列和使用区间。
- 1-3 秒短镜头是否通过序列裁切、静态图动效或后期变速解决。

### 主体一致性

- 脸、发型、服装、体型、材质是否稳定。
- 关键道具、品牌标识、颜色是否漂移。
- 多镜头之间角色是否像同一个人或同一个物体。

### 动作执行

- 主体是否完成分镜要求的主动作。
- 动作是否自然，是否有僵硬、漂浮、穿模。
- 手部、脚步、车辆、动物、机械结构是否可信。
- 是否把多个动作混成不可理解的运动。

### 镜头语言

- 景别是否正确。
- 构图是否服务主体。
- 运镜是否符合要求。
- 是否有突兀跳切、镜头乱晃、主体出画。

### 场景和风格

- 场景是否符合时代、地点、天气和光线。
- 风格是否稳定，是否出现游戏 CG 感、塑料质感、过度磨皮。
- 背景是否有明显乱码文字、畸形建筑或异常物体。

### 声音和字幕

- 对白是否清晰，是否与画面冲突。
- 环境声和音效是否符合动作。
- 字幕是否可读，是否有乱码、错字、遮挡主体。
- 如果模型音频不稳定，应建议后期单独制作声音。

### 平台适配

- 前 1 秒是否有明确主体或冲突。
- 前 3 秒是否能留住观众。
- 是否适合竖屏裁切。
- 是否有封面可截取画面。
- 是否存在平台合规风险。

## 重生成建议写法

不要只说“重新生成”。要指出改哪一类变量：

```text
问题：角色脸部在第 3 秒后漂移。
建议：保留原提示词主体和场景，减少动作复杂度，增加“保持同一张脸、同一发型、同一服装”，改为 3 秒近景单动作镜头。
```

```text
问题：镜头没有执行推近，而是随机摇晃。
建议：把运镜改成“slow dolly in, stable camera, no handheld shake”，删掉其他镜头动作。
```

```text
问题：画面有大量乱码文字。
建议：删除“报纸标题、招牌文字”等复杂文字要求，改为“unreadable generic signage”或后期叠加真实文字。
```

```text

1. {问题，出现位置，影响}
2. {问题，出现位置，影响}

符合项：
- {主体/动作/镜头/风格中做得好的部分}

修改建议：
- {提示词怎么改}
- {分镜怎么拆}
- {后期怎么补}

下一步：
- {直接剪辑 / 重新生成镜头 X / 补参考图 / 改模型}
```
