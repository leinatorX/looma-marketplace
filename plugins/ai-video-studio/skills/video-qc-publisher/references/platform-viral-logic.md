# AI 视频表现形式与平台逻辑

## AI 视频表现形式

| 类型 | 适合内容 | 制作重点 |
| --- | --- | --- |
| 文生视频 | 无素材创意、视觉奇观、概念片 | 提示词必须完整描述主体、动作、场景和镜头 |
| 图生视频 | 角色一致、产品展示、封面延展 | 保持参考图主体，重点写运动和镜头 |
| 视频延展 | 延长已有镜头、补转场 | 保持前后帧风格和动作连续 |
| 重混改写 | 换风格、换场景、换情绪 | 明确保留项和替换项 |
| 多参考生成 | 角色、场景、风格分离控制 | 标清每个参考素材的用途 |
| 口播/虚拟人 | 知识、观点、带货、课程 | 口型、字幕、语速和人设一致性 |
| 剧情短片 | 悬疑、反转、情绪故事 | 强钩子、短镜头、冲突递进 |
| 产品广告 | 卖点、质感、转化 | 痛点、功能演示、结果证据 |
| 科普可视化 | 抽象知识、流程、事故复盘 | 概念可视化、字幕清楚、节奏稳定 |
| 伪纪录片/监控 | 真实感、异常事件、恐怖悬疑 | 监控视角、时间戳、低干预镜头 |
| 循环视频 | Shorts、TikTok、视觉爽感 | 结尾能自然回到开头 |

## 通用爆款逻辑

- 前 1 秒要出现明确主体、冲突、结果或反差。
- 前 3 秒不要解释背景，先给“为什么要看”。
- 中段每 2-4 秒给一次视觉变化、信息递进或情绪变化。
- 结尾要有反转、答案、强画面、行动号召或循环点。
- 标题、封面、首帧、字幕必须服务同一个钩子。
- AI 视频不要只靠画面奇观，必须有可复述的看点。

## 平台适配

### 抖音 / TikTok

适合：

- 强反差、强动作、强情绪、强节奏。
- 视觉奇观、剧情反转、产品前后对比。

写法：

```text
开头直接给结果或冲突；镜头短；字幕强；情绪明确；结尾制造评论点或循环点。
```

### 小红书

适合：

- 种草、经验、审美、人设、生活方式、教程。
- 画面要更真实、更干净、更有“可参考价值”。

写法：

```text
封面标题先给利益点；内容强调真实体验、步骤、对比和个人感受；避免过度广告腔。
```

### B站

适合：

- 信息量、完整解释、系列化内容、创作过程、技术拆解。
- AI 视频可以展示幕后、分镜和制作逻辑。

写法：

```text
标题可有悬念但要兑现；内容逻辑完整；适合加入制作过程、失败对比和知识解释。
```

### 视频号

适合：

- 价值感、情绪共鸣、知识科普、熟人转发。
- 适合更稳重的表达，不宜只追求猎奇。

写法：

```text
强调实用、可信、情绪共鸣和可转发理由；封面文案要清楚直接。
```

### YouTube Shorts

适合：

- 全球化视觉、循环观看、强节奏、清晰字幕。
- 不依赖复杂中文语境的内容更容易跨语言传播。

写法：

```text
画面自解释；字幕短；结尾可无缝回到开头；标题清楚表达冲突或结果。
```

### 快手

适合：

- 真实生活、人情味、技能、地方性、实用经验。

写法：

```text
少一点炫技，多一点真实场景和人物关系；适合朴素但有情绪的叙事。
```

## 风险提示

不要把平台逻辑写成保证爆款。只能说“更符合平台常见推荐和留存逻辑”。涉及广告、医疗、金融、教育、招聘、未成年人、危险动作时，需要单独做合规检查。
